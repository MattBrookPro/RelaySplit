RelaySplit

Control plane — checking…

Network-delivered real-time GPU audio processing. This service is the control plane: it handles signalling, presence and ephemeral TURN credentials. Audio never passes through it — that's the data plane (WebRTC, peer-to-peer or TURN-relayed).

  • WebRTC signalling: wss://relaysplit.vaguelystrange.com/ws
  • Ephemeral TURN: POST /api/turn
  • Presence: GET /api/channels
